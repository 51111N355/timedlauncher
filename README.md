# TimedLauncher
LaunchWrapper с добавленым измерением времени трансформации за время существования процесса.

Измерения всех трансформаций (и количество) пишутся в лог при закрытии Java процесса (из shutdownHook).


## Измерения запуска на сборках
Количество классов + общее время трансформаций с момента запуска игры, до момента открытия главного меню.

В момент открытия главного меню я сразу нажимал кнопку Quit.

Условия измерений:
- Лаунчер: MultiMC
- Выделено: 4GB памяти игре (4096 MB) из 16 в системе
- Процессор: AMD Ryzen 7 7435HS
- Видеокарта: RTX 3050 Laptop
- Диск: NVME SSD SAMSUNG MZVL8512HELU-00BTW

## Точность измерений
Тест на том же устройстве показал, что даже 10,000,000 выполнений цикла с 2 вызовами в System.nanoTime() занимает около 1 секунды.

Время выполнения одного вызова ~= 50нс - в таком случае за условные 40000 вызовов - ошибка измерения будут около 2мс.

```java
public class Main {
    public static void main(String[] args) {
        long start = System.nanoTime();

        int CALLS = 10_000_000;
        for (int i = 0; i < CALLS; i++) {
            long a = System.nanoTime();
            long b = System.nanoTime();
        }

        long end = System.nanoTime();
        System.out.println("Time: " + (end - start)); // Time: 1008462800
    }
}
```
